                  # 1.1 Redis的安装

                  # 下载源码

                   wget https://download.redis.io/redis-stable.tar.gz

                  # 编译Redis

                  // 解压
tar zxvf redis-stable.tar.gz

// 编译之前需要安装make和gcc
sudo apt install make
sudo apt-get install gcc
sudo apt install pkg-config

// 编译
make
